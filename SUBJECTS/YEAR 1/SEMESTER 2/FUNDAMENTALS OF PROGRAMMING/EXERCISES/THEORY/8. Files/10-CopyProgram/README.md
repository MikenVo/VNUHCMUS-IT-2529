# <p align="center"> Problem 10 - Files </p>

## Problem
Write a program to copy a file to another file.

## Test Case
### Input
file1.txt
```
Hello, my name is Mike, I am a first-year student studying Information Technology at Ho Chi Minh City University of Science.
I come from Vietnam.
```
  
file2.txt
```
I live in Ho Chi Minh City.
I love Computer Science and Engineering.
```

### Output
```
Hello, my name is Mike, I am a first-year student studying Information Technology at Ho Chi Minh City University of Science.
I come from Vietnam.
I live in Ho Chi Minh City.
I love Computer Science and Engineering.
```

## Note
You can test by running the code, remeber to create the files before executing the program.
