We need to get the inputs of a, b and c.
After that, check if a is equal 0.
    If yes, the equation will become a linear equation, so we will use the linear equation solver.
    If no, do the steps below.

First, calculate the delta = b*b - 4*a*c.
Second, check the delta.
    If delta < 0, print "No solution"
    If delta == 0, print (-b)/(2*a)
    If delta > 0, print (-b + sqrt(delta))/(2*a) and (-b - sqrt(delta))/(2*a)
