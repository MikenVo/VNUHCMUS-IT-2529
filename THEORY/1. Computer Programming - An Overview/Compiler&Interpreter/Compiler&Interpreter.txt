1. Compiler
- Well-known programming languages that use a compiler: C/C++
- A compiler is a software that compile the source code from human language to computer language (0s and 1s).
    - Compiler compile all the source code once before executing the program.
    - The compile time is long depends on the number of lines.
    - The program after compiling is run much faster because the computer works much easier
    with a source code translated into computer language.
2. Interpreter
- A well-known programming language that use a interpreter: Python
- An interpreter is also a software that interpret the source code from human language to computer language.
    - Interpreter interpret the source code line by line for the computer to do it line by line too.
    - The interpreter time is much faster than the compiler.
    - The program is run much slower because the source code has to be interpreted line by line over and over.
