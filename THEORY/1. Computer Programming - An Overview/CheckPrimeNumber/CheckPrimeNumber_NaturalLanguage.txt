    Define "num" variable.
    Define "n" variable and set 0 to "n"

    Initialize an infinite loop
    Let the user prompt "num"
    Check if "num" is greater than 1.
        If true,
            Initialize a for loop, i starts at 1 and stop if i > num.
            Check if num % i == 0.
                If true, increase i by 1.
                If false, ignore.
            
            Check if n == 2.
